Basic UNIX commands
#Command Args Option Description command fullname
cd destination directry change directry change directry
ls destination directory lists your files list
-l lists your files in 'long format', which contains lots of useful information, e.g. the exact size of the file, who owns the file and who has the right to look at it, and when it was last modified.
-a lists all files, including the ones whose filenames begin in a dot, which you do not always want to see.
mv filename1 filename2 moves a file (i.e. gives it a different name, or moves it into a different directory (see below) move
cp filename1 filename2 copies a file copy
filedir1 filedir2 -r copies a directory
rm filename removes a file. It is wise to use the option rm -i, which will ask you for confirmation before actually deleting anything. You can make this your default by making an alias in your .cshrc file. remove
filedir -r remove a directory
filename -f remove a file force
diff filename1 filename2 compares files, and shows where they differ different
mkdir dirname make a new directory make directory
pwd tells you where you currently are. print work directory
