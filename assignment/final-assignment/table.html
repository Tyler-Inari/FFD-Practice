-a Include dir entries whose names begin with a dot (‘.’).
-l Follow all symbolic links to final target and list the file or dir the link references rather than the link itself. This option cancels the -P option.
mv source target move files (rename files) move
source ... dir move files

source_dir target_dir -r copy a dir

file ... -f remove files force
